Detour: Writing C# Code in Neovim

I decided to take a brief detour from working on the breakout clone to focus on improving my editor. From what I can tell, Visual Studio is the IDE of choice for most C# developers, but I'm a masochist apparently and prefer to use Neovim.

One of the nice things about using an IDE like VisualStudio is that it will tell you very quickly when you have something wrong with your code that will prevent it from executing the way you expect, or even just compiling.
